🔧 Service Worker Manager

Agar app offline issue aa raha hai, toh yeh tools use karo:

Manual Steps:

  1. Browser DevTools kholo (F12)
  2. Application tab me jao
  3. Service Workers section me jao
  4. "Unregister" button click karo
  5. Cache Storage me jao aur sab delete karo
  6. Page reload karo (Ctrl+Shift+R)
